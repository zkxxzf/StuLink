# StuLink — 智联校园学生管理系统 · 用户手册

## 快速开始

### 访问地址
```
http://服务器IP:5000
```

### 默认管理员
```
用户名: admin
密码:   admin123
```
> ⚠️ 首次登录后请立即修改密码！

### 部署方式

系统支持 Docker 全镜像一键部署和阿里云服务器部署，详见 [部署文档](部署文档.md)。

**Docker 部署（最快）**：
```bash
docker load -i stulink-v1.5.0.tar
docker compose up -d
```

## 二、登录与基本操作

### 2.1 登录
浏览器打开系统地址 → 输入用户名密码 → 点击"登录"。

## 三、宿舍管理模块

左侧菜单 → **宿舍管理**（或欢迎页点击"宿舍管理"卡片），进入宿舍仪表盘。

### 3.1 侧边栏导航
进入宿舍管理后，左侧显示宿舍专属菜单：
- 学生查询
- 宿舍列表 / 分配宿舍 / 宿舍报表
- 床位分配
- 宿舍总览
- 统计报表

### 3.2 学生查询
左侧菜单 → **学生查询**，支持按姓名、学号、年级、班级、性别组合搜索。

### 3.3 宿舍列表
查看所有宿舍，按楼栋/性别/楼层/床位数筛选。

### 3.4 新增宿舍
点击"新增宿舍" → 选择楼栋、房间号、性别、床位数 → 保存。支持批量新增。

### 3.5 分配宿舍给班级
- **可视化分配**: 拖拽班级到宿舍房间
- **自动分配**: 选择班级+房间 → 系统计算最优方案 → 预览 → 执行

### 3.6 宿舍报表
查看按年级/性别/班级组织的分配总览，支持导出 Excel。

### 3.7 床位分配
选择年级班级 → 拖拽学生到空床位。支持一键自动分配、搬移、互换。

**限制**: 性别必须匹配、非合班宿舍仅限本班学生。

### 3.8 统计报表
全校/年级/班级三级视图，展示男女比例、住校走读分布。

## 四、系统管理模块

左侧菜单 → **系统管理**（或欢迎页点击"系统管理"卡片），进入学生列表。

> 侧边栏显示系统专属菜单：学生管理、用户管理、字典管理、班型设置、权限组管理、年级管理。
> 无权用户点击受限页面会提示"无权限"。

### 4.1 学生管理
学生的增删改查、批量导入、批量调班。

**新增学生**: 填写姓名、性别、学号（必填）→ 选择年级班级 → 保存。  
**批量导入**: 下载 Excel 模板 → 填写数据 → 上传。必填列：姓名、性别、**学号**、身份证号。学号为跨模块业务主键，必须填写且唯一。  
**批量删除**: 仅管理员可用。

### 4.2 用户管理
新建/编辑用户，设置角色（管理员/宿管教师/班主任/年级长/全校组），重置密码，启用/禁用。

### 4.3 字典管理
管理下拉选项：年级、班级、楼栋、楼层、走读类型等。被引用的值无法删除。

### 4.4 班型设置
设置各班班型、选科方向，用于自动分配时的合班判断。

### 4.5 权限组管理
创建自定义权限组，控制菜单可见范围和数据范围。

### 4.6 年级管理
**毕业操作**: 清空该年级的宿舍和床位分配，同时备份主数据库（`dormitory.db`）和分配历史库（`history.db`）到 `data/backups/`，方便日后追溯。  
**历史查询**: 查看历史毕业备份数据。

### 4.7 寝室分配历史
系统自动记录每次床位分配、取消、搬移、互换、清空操作。记录保存在独立的历史库中，包含学生快照、房间信息、操作人、学期等字段，即使学生被删除仍可追溯。每学期重新分寝时的历史数据永久保留。

## 五、常见问题

**Q: 忘记管理员密码？**  
A: 运行 `python scripts/init_db.py`（会覆盖字典数据，谨慎使用）。

**Q: 身份证号显示为空？**  
A: AES 加密存储，旧明文数据自动兼容读取。

**Q: 自动分配报"床位不足"？**  
A: 扩大房间选择范围或启用极限模式（8人间满8人）。

**Q: 导入 Excel 报"学号为空"？**  
A: 学号是跨模块业务主键，不能为空。请确保每个学生都有学号。

**Q: 导入 Excel 报"身份证号格式不正确"？**  
A: 身份证号必须 18 位，校验码正确。

**Q: 每学期重新分寝后，之前的分配记录还在吗？**  
A: 在。系统自动在独立历史库中记录所有分配变更，即使重新分寝也不会丢失。毕业操作也会备份历史库。
